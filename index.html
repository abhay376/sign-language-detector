How the Sign Language Detector Works

1

Webcam Capture

The browser accesses your webcam via WebRTC and captures live video frames using an HTML5 Canvas.

2

Hand Landmark Detection

Each frame is sent to the FastAPI backend where Google MediaPipe detects 21 hand landmarks in 3D space.

3

ML Classification

The landmark coordinates are fed into a custom-trained Random Forest classifier (scikit-learn) which predicts the ASL letter.

4

Real-Time Display

The prediction and confidence score are returned as JSON and displayed instantly on the web interface.
